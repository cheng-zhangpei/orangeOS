# ch3：批处理系统与分时系统

锯齿螈：多道程序装载，这里每一个App都是分开的可以独立去进行系统调用。并且发现os的镜像是和系统调用分开的，也就是没有利用链接器。

![锯齿螈多道程序操作系统 -- Multiprog OS总体结构](https://rcore-os.cn/rCore-Tutorial-Book-v3/_images/jcy-multiprog-os-detail.png)

> 我之前其实没有注意，rCore将不同的时期的os命名为生物的进化过程。哈哈哈现在是锯齿螈

批处理系统的特性是：在内存中同一时间最多只需驻留一个应用。这是因为只有当一个应用出错或退出之后，批处理系统才会去将另一个应用加载到相同的一块内存区域。
